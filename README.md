# Sistema-de-Controle-de-Estoque
Sistema de Controle de Estoque desenvolvido em Java e MySQL. O projeto permite cadastrar, listar e excluir produtos, além de calcular o valor total do estoque. Desenvolvido com foco na aplicação de Programação Orientada a Objetos (POO), JDBC e operações CRUD, promovendo a integração entre aplicação Java e banco de dados relacional.
